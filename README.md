# Бур / Бурик — Telegram мем-бот

Мемный бот для групповых чатов Telegram на `aiogram`.
Бот читает сообщения в чате, хранит недавний контекст и генерирует абсурдные ответы.

## Структура проекта

```text
.
├── bot.py              # Точка входа, запуск polling, логирование, глобальная обработка ошибок
├── handlers.py         # Обработчики команд, реакции на упоминания/случайные триггеры, антиспам
├── generator.py        # Генератор мемных и псевдофилософских фраз
├── config_manager.py   # Персистентные настройки чатов (JSON)
├── requirements.txt    # Зависимости
└── README.md           # Документация
```

## Функции

- Работа в `group/supergroup`
- Реакция на:
  - команды
  - упоминание `@botname`
  - случайные события (по шансам из конфигурации)
- Анализ последних сообщений (по умолчанию 10, настраиваемо 5..20)
- Режим хаоса:
  - повышает спонтанные ответы
  - искажает отдельные слова в ответах
- Отдельная конфигурация для каждого чата:
  - `random_reply_chance`
  - `analysis_window`
  - `chaos_mode`
- Логирование
- Базовая защита от спама (cooldown на ответы)
- Глобальная обработка ошибок

## Команды

- `/бур` — анализ последних сообщений и мемный вывод
- `/бурик` — случайная абсурдная фраза
- `/режим` — показать текущий режим
- `/настройки` — показать конфигурацию чата
- `/хаос` — переключить chaos-режим

## Установка и запуск

1. Установить Python 3.10+
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Задать токен бота:

```bash
export BOT_TOKEN="<ваш_токен_бота>"
```

4. Запустить:

```bash
python bot.py
```

## Как это работает

- Все входящие текстовые сообщения (кроме сообщений ботов) в группах сохраняются в буфер истории чата.
- При `/бур`, упоминании бота или случайном триггере берутся последние `analysis_window` сообщений.
- Генератор выбирает ключевые слова, смешивает их с шаблонами и формирует короткий мемный ответ.
- Конфиги чатов автоматически создаются и сохраняются в `chat_configs.json`.
